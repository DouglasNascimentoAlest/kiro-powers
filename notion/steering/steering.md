# Criar Documentação
- Organize páginas em hierarquia lógica
- Use headings para estruturar conteúdo
- Adicione blocos de código com syntax highlighting
- Inclua callouts para informações importantes

# Gerenciar Databases
- Defina propriedades apropriadas para o tipo de dado
- Use filtros para consultas específicas
- Mantenha opções de select consistentes
- Relacione databases quando necessário

# Sincronizar com Código
- Atualize docs quando APIs mudam
- Mantenha changelog atualizado
- Documente decisões arquiteturais
- Vincule código a documentação

# Colaboração
- Use comentários para discussões
- Mencione pessoas em tarefas
- Mantenha status atualizados
- Compartilhe páginas relevantes com stakeholders
